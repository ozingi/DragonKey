# DragonKey
## DragonKey是什么？
DragonKey是用.NET MAUI写的跨平台应用，已适配Android和windows端，均支持黑暗模式。是一个自托管2FA认证应用，比如代替Google身份验证器给GitHub、X、meta、Binance、OKX等的二次认证。但未来DragonKey将不仅仅是密码管理器...
